# smart-warehouse-inventory-sap-abap
SAP ABAP CDS and ABAP SQL project demonstrating joins, associations, aggregations, GROUP BY and HAVING.

## Project Overview

This SAP ABAP project demonstrates:

- CDS Views
- Associations
- Composition
- Parent-Child Relationship
- Joins
- Calculated Fields
- ABAP SQL
- GROUP BY
- HAVING
- Aggregate Functions

## Technologies

- SAP ABAP Cloud
- CDS Views
- ABAP SQL
- Eclipse ADT

## Database Tables

### ZMAT_MASTER_MUT

Material Master Data

### ZSTOCK_ITEMS_MUT

Warehouse Stock Data

## Features

- Material Management
- Warehouse Inventory Tracking
- Stock Value Calculation
- Quantity Aggregation
- Average Price Analysis
